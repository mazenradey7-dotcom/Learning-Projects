# مشروع Data Science - تحليل بيانات الطلاب

## الوصف
هذا المشروع البسيط يوضح كيفية استخدام **Python وPandas** لتحليل بيانات الطلاب، مثل الدرجات والمعدل العام.

## البيانات المستخدمة
- ملف CSV يحتوي على أسماء الطلاب، المواد، والدرجات.
- مثال:
| الاسم       | الرياضيات | الفيزياء | اللغة الإنجليزية |
|------------|-----------|---------|----------------|
| مازن       | 85        | 78      | 90             |
| علي        | 70        | 88      | 75             |
| سارة       | 95        | 92      | 89             |

## الأهداف
- حساب المعدل لكل طالب.
- معرفة أعلى وأدنى درجة في كل مادة.
- رسم بياني لمقارنة أداء الطلاب.

## الكود (Python)
```python
import pandas as pd

# قراءة الملف
df = pd.read_csv('students_scores.csv')

# حساب المعدل لكل طالب
df['المعدل'] = df[['الرياضيات','الفيزياء','اللغة الإنجليزية']].mean(axis=1)

# أعلى وأدنى درجة في كل مادة
max_scores = df.max()
min_scores = df.min()

print(df)
print("أعلى الدرجات:\n", max_scores)
print("أدنى الدرجات:\n", min_scores)
